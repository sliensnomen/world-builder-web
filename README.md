# WorldWeaver · 织界

世界观规则引擎。不是笔记软件，不是 AI 工具。

让设定自己记住自己。

## 关于

WorldWeaver 是一款面向桌面端的世界观一致性管理工具，基于 SwiftData 声明式约束构建，零 AI、零网络即可运行。它帮助创作者在宏大的设定体系中自动检测矛盾、追踪关系路径，并保持设定集的内在一致性。

## 技术栈

- Next.js 16
- React + TypeScript
- Tailwind CSS 4
- Framer Motion

## 本地运行

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看页面。

## 构建

```bash
npm run build
```
